# System Architecture

## Overview

AI Stadium Copilot follows a scalable cloud-native architecture.

## Components

### Frontend

Next.js application providing:

- Fan dashboard
- Organizer dashboard
- Volunteer dashboard
- AI assistant interface


### Backend

FastAPI services handling:

- Authentication
- Business logic
- AI orchestration
- Data processing


### AI Layer

Google Gemini provides:

- Conversational AI
- Summarization
- Recommendations
- Multilingual assistance


### Database

PostgreSQL stores:

- Users
- Stadium information
- Events
- Reports
- Analytics
